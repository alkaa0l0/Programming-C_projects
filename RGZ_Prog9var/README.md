# translate — простейший переводчик (РГР, вариант 9)

Переводит текст по словарю, сохраняя форматирование (пробелы, переносы строк,
знаки препинания). Замена — только по полному совпадению слова.

## Что реализовано (оценка «5» + «Отлично»)
- Словарь из файла, динамическая память (массив статей).
- Структуры (Entry, Dictionary), сохранение регистра, многозначные слова.
- Проверка входных данных с понятными ошибками.
- Сборка через CMake, UNIT-тесты (CTest).
- Логирование в файл: INFO / WARNING / ERROR.
- Конфигурационный файл формата KEY=VALUE.

## Сборка
    mkdir build && cd build
    cmake ..
    make

## Запуск
    # три файла: вход, словарь, выход
    ./translate ../examples/input.txt ../examples/dict.txt out.txt

    # через конфиг (запускать из корня проекта):
    cd ..
    ./build/translate --config examples/config.ini

## Опции
    --config FILE      файл настроек (KEY=VALUE)
    --log-level LEVEL  INFO | WARNING | ERROR
    --no-case          не сохранять регистр
    --alternatives     показывать значения многозначных слов
    --help             справка

## Тесты
    cd build && ctest --output-on-failure

## Формат словаря
    привет = hello
    мир    = world | peace     # многозначное слово (значения через '|')
