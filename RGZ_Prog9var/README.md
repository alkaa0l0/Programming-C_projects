# translate — простейший переводчик (РГР, вариант 9)

Программа переводит текст по словарю, сохраняя форматирование (пробелы, переносы
строк, знаки препинания). Замена выполняется только по полному соответствию слова.

## Возможности
- Словарь из файла, динамическая память (хеш-таблица).
- Структуры данных, сохранение исходного регистра, многозначные слова.
- Проверка всех входных данных с понятными сообщениями об ошибках.
- Сборка через CMake, UNIT-тесты (CTest).
- Логирование в файл с уровнями INFO / WARNING / ERROR.
- Конфигурационный файл формата KEY=VALUE.
- Усложнение: многопоточность (pthread).

## Сборка
```bash
mkdir build && cd build
cmake ..
make
```

## Запуск
```bash
# Через позиционные аргументы:
./translate <вход.txt> <словарь.txt> <выход.txt> [опции]

# Через конфигурационный файл:
./translate --config ../examples/config.ini

# Пример:
./translate ../examples/input.txt ../examples/dict.txt out.txt --alternatives
```

### Опции
| Опция            | Назначение                                      |
|------------------|-------------------------------------------------|
| `--config FILE`  | файл конфигурации (KEY=VALUE)                    |
| `--log-level L`  | уровень лога: INFO \| WARNING \| ERROR           |
| `--threads N`    | число потоков (>=1)                              |
| `--no-case`      | не сохранять регистр исходного слова             |
| `--alternatives` | показывать все значения многозначных слов        |
| `--help`         | справка                                          |

## Тесты
```bash
cd build
ctest --output-on-failure
```

## Формат словаря
```
# комментарий
привет = hello
мир    = world | peace      # многозначное слово (значения через '|')
```
